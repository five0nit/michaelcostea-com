# Button Kit (michaelcostea.com)

Use these icon assets as button graphics in `styles.css` or by linking:

```html
<link rel="stylesheet" href="./assets/pixel-pack/button-kit.css">
```

## CSS class pattern

- Base: `ui-btn`
- It works whether or not the legacy `btn` class is also present.
- Optional: `icon-only`
- Icon variants below: `icon-welcome`, `icon-resume`, `icon-projects`, `icon-ai-agents`, `icon-ai-help`, `icon-contact`, `icon-dashboard`, `icon-review`, `icon-workflow`, `icon-radar`, `icon-reports`, `icon-knowledge`, `icon-automation`, `icon-journey`, `icon-safe`, `icon-strategy`, `icon-terminal`, `icon-fields`, `icon-files`, `icon-recycle`
- Optional alignment utilities:
  - `uniform` / `full` / `btn-full` => force full-width, centered content

## Example usage

```html
<button class="ui-btn icon-welcome">Launch Welcome</button>
<button class="ui-btn icon-only icon-resume" aria-label="Open Resume"></button>
<a class="ui-btn icon-projects" href="#">Open Projects</a>
```

How to add a new ask game

  • Create ask-games/<slug>/unreal/ and add your Unreal web export there.
  • Create ask-games/<slug>/index.html (optional) to run a launcher that redirects to unreal/index.html.
  • Point homepage/project links to ask-games/<slug>/.

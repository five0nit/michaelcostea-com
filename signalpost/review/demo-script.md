# SignalPost app-review demo script

## Recording rules

- Capture one continuous end-to-end browser recording.
- Use a dedicated TikTok test account permitted by TikTok's sandbox/review process.
- Keep the test account private while the app remains unaudited and use `Only me`.
- Do not expose passwords, OAuth codes, provider credentials, browser developer tools, local paths, or unrelated tabs.
- Use an original test MP4 with no SignalPost watermark, unrelated brand mark, third-party clip, or unlicensed audio.

## Script

1. Open `https://michaelcostea.com/signalpost/`.
2. Show the product name, description, public audience, and visible Privacy, Terms, and Support links.
3. Open each legal/support URL briefly, then return to the app.
4. Press **Connect TikTok**.
5. Show TikTok's authorization screen and requested scopes.
6. Approve with the dedicated test creator.
7. Return to SignalPost and show the connected nickname, username, and avatar.
8. Show that privacy begins at **Choose privacy**, not a preset value.
9. Select the original MP4 and play several seconds of the preview.
10. Type and edit a caption.
11. Select **Only me** for the unaudited/private test flow.
12. Toggle Comments, Duet, and Stitch where the creator account permits them; show disabled provider controls if any.
13. Turn Commercial Content Disclosure on without selecting a category; show that publishing remains disabled.
14. Select **Your brand**; show the “Promotional content” label notice.
15. Select **Branded content** while privacy is private; show the option is blocked and the private-visibility warning.
16. Show the Music Usage Confirmation link and wording.
17. Check Music Usage Confirmation and explicit publishing consent.
18. Press **Publish to TikTok** once.
19. Show initialization/processing status and explain that a publish ID is not claimed as public proof.
20. Show that both confirmation boxes reset after the attempt.
21. Use **Disconnect and delete data**.
22. Show separate local-deletion and provider-revocation status.

## After TikTok approves Direct Post

Repeat the recording using a permitted public/friends setting. Show the server gate enabled only after approval, complete one controlled post, poll to terminal provider success, and verify the resulting post on the exact connected profile.

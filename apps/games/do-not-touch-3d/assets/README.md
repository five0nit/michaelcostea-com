# Custom 3D Assets

This folder is reserved for the original DO NOT TOUCH 3D asset pipeline.

Planned structure:

- `models/` — custom GLB/GLTF character, NPC, prop, and environment assets
- `textures/` — hand-authored textures / atlases
- `ui/` — journal icons, quest markers, interface frames
- `audio/` — music, SFX, interaction cues

The current MVP still uses placeholder in-engine geometry.
Use `data/assets.json` as the replacement manifest when custom assets are produced.
